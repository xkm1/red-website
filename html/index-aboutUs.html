  • 网站介绍

    该网站有四个主题，跨越时空的回信，工农武装，红色读本，耀邦陵园。主题一是为了学习《跨越时空的回信》用传承的信仰，致敬先烈。主题二是为了学习江西数字展馆，发扬红色文化。主题三是为了回顾红色历史，学习红色文化。主题四是红色实地走读去的耀邦陵园，为了纪念胡耀邦同志。四个主题响应“奋进新征途，建功新时代”主旋律，切实激发大学生向第二个百年奋斗目标前进。

  • 插件使用

    该网站用到了两个插件，rellax 和 particles.js，前者可以快速开发滚轮响应视差效果，后者可以添加一个粒子特性背景。Rellax 是一个黄油般光滑、超轻量级的 javascript 视差库。插件上的使用是非常方便的，只需要通过设置属性data-rellax-speed就可以完成视差效果。particles.js是一个用于粒子背景的轻量级、无依赖和响应式 javascript 插件。该插件在使用起来也是极其方便的，只需要在一个元素中添加id="particles-js"就可以实现一个粒子特性背景。同时该插件也有非常多的属性可以设置，"color"属性可以设置粒子的颜色，"line_linked"可以设置粒子是否链接，"shape"可以设置粒子的形状，比如圆形，三角形，正方形等等。

  • 学习过程

    为了完成该网站，前前后后学习了2个月，技术栈从html到css再到javascript，全部都是自学后完成的。在网站的开发中，由于是第一次开发网站，所以遇到过许多大大小小的问题，大到页面的布局，小到z-index等层级问题。在开发过程中，也学习了许多其他网站的优秀作品，同时别的作品也用到许多别的技术栈，比如css的扩展语言scss和html的模板语法pug等等，为此也在不断的学习来看懂代码。页面的布局和美化也是开发过程中的一大难点，因为互联网中并没有多少可以借鉴的个人制作的优秀红色网站，所以只能在外网学习优秀作品的布局来制作自己的网站。好在最后是完成了该网站的开发，但这些是远远不够的，在未来的时间里，我会进一步优化自己的网站，添加更多的内容和功能，并作为个人博客上线。
